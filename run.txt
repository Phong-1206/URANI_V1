# URAIN_V1
Đây là phiên bản Ddos đầu của TRẦN TUẤN PHONG còn nhiều sai xót có gì thông cảm nhé.
Lệnh chạy: 
git clone https://github.com/Pvfiwv/URAIN_V1
cd URAIN_V1
unzip URAIN_V1.2.zip
Nếu nó ra folder thì cd vào folder đó nhé
Còn không thì thì chạy lệnh nhé...
python setup.py
python main.py
Thiếu thư viện nào tự pip install nhé.
Tốt nhất nên chạy cho gg cloud nhé, termux có nhiều thư viện không cài được.
